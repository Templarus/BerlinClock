К сожалению,  я  посчитал,
что целью задания было разработать графическое отображение берлинских часов (достаточно гибкое  и конфигурируемое для использования в разных ситуациях).

Класс clock имплементирует интерфейс Timeconverter и его метод convertTime.
Метод calculateConvertedTime выполняет конвертацию полученного string в текстовый вариант "берлинских часов". 
Все остальные методы нужны для реализации графического интерфейса.
